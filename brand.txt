# FiveWell Africa Brand Guidelines

## Color Palette

### Primary Colors
- Midnight Blue: #1d3a72 (Primary)
- Sandstone: #f6eedf (Primary Background)
- White: #ffffff (Text on Dark)

### Secondary Colors
- Sage Green: #8a9b6e (Secondary)
- White: #ffffff (Secondary Background)

## Typography
- Primary Font: Lato
  - Weights: 300, 400, 700, 900
  - Styles: Regular, Italic, Bold

## Usage Notes
- Use Midnight Blue for primary actions and important elements
- Sandstone serves as the primary background color
- Sage Green for secondary actions and accents
- White for text on dark backgrounds and clean spaces

## Color Variables (for reference)
```css
:root {
  /* Primary Colors */
  --primary: #1d3a72;
  --primary-foreground: #f6eedf;
  
  /* Secondary Colors */
  --secondary: #8a9b6e;
  --secondary-foreground: #ffffff;
  
  /* Backgrounds */
  --background: #ffffff;
  --foreground: #1d3a72;
  
  /* Additional Colors */
  --muted: #f1f5f9;
  --muted-foreground: #64748b;
  
  /* Border */
  --border: #e2e8f0;
  --input: #e2e8f0;
  --ring: #1d3a72;
}
```

## Font Stack
```css
font-family: 'Lato', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif;
```

Note: Please ensure the Lato font is properly imported in your project.
